# YouTube Clone (MERN Stack)

## Project Overview
This project is a **YouTube Clone** built using the **MERN stack** (MongoDB, Express.js, React.js, Node.js).  
Users can **browse videos, search, filter by category, register/login, watch videos, comment, like/dislike**, and manage their **channels and uploaded videos**.  

---

## Features

### Frontend
- **Home Page**:  
  - Displays video thumbnails with title, views, and channel name  
  - Filter buttons for categories  
  - Search bar functionality  
- **Video Player Page**:  
  - Watch video  
  - Add, edit, delete comments  
  - Like/Dislike functionality  
- **Channel Page**:  
  - Create, edit, delete videos  
  - View uploaded videos  
- **User Authentication**:  
  - Register/Login with JWT authentication  
- **Responsive Design**:  
  - Works on desktop, tablet, and mobile  

### Backend
- **Node.js + Express API**  
- **MongoDB** database for storing users, videos, channels, comments  
- **JWT Authentication** for secure routes  
- CRUD operations for videos, channels, and comments  

---

## Tech Stack

- **Frontend**: React, Axios, React Router  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB (Atlas or Local)  
- **Authentication**: JWT  
- **Version Control**: Git & GitHub
